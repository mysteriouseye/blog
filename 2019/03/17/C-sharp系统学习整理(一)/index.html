3 C#的字符串与输出

所有编程语言都逃不过Hello world魔咒，C#也不例外
我们来看一段c#标准的hello world吧！

    using System;   //使用系统库
    namespace demo1   //命名空间（文件夹名字）
    {
        class Program  //类名
        {
            static void main(String[] args)  //程序运行入口
            { 
                Console.WriteLine("Hello world"); //输出
            }
        }
    }

很简单，不多说吧。如果学过c艹和java的会发现c#两个都有点像，没错 它就是汲取了c艹和java的一些语法特点，特别像java，却胜过java，，，（人类本质是复读机）

可能你会觉得很简单，输出不跟java差不多嘛，这就错啦，c#的输出系统可是略强一丢丢的

例如下面

    using System;
    namespace demo2
    {
        class Program
        {
            static void main(String[] args)
            {
                String one = "Mike";
                Console.WriteLine($"Hello {one}");
            }
        }
    }

看到变化没？字符串前面多了个美刀符，然后上面多了个字符串类型的变量声明，它的值是Mike，而hello后面的，{one} 代表什么？
输出结果，如果你猜是Hello {one}，那就是错的！实际输出是Hello Mike，这个玩意，微软官方称为字符串内插

相对于java，java的System.out是没有此功能的，你想要内插一个变量到一段字符串，你还得用啥子StringBuilder 啊，然后声明的StringBuilder变量调用插入，还得写个插入的位置，而且还不止这个方法。。。接着你还得toString一下，噗。。。
c#真优雅~

可能由娃子不服，说system.out不是可以用+号嘛？

那么下面的呢？

using System;

namespace demo1
{
    class Program
    {
        static void Main(string[] args)
        {
            String aFriend = "Maria";
            String secondFriend = "Sage";
            Console.WriteLine($"Hello World! {aFriend}");
            Console.WriteLine($"My friends are {aFriend} and {secondFriend}");
            Console.WriteLine($"The name {aFriend} has {aFriend.Length} letters.");
            Console.WriteLine($"The name {secondFriend} has {secondFriend.Length} letters.");
        }
    }
}

这里用了Length，这可是计算出整数的，和+号意义就不一样了，明显比java的system.out方便的多。

下面还有一个挺不错的自带方法，很省事，虽然说对研究本质没帮助，但是我觉得你可以自己去实现一遍呀~都是想偷懒的借口

using System;
namespace demo02
{
    class Program
    {
        static void Main(string[] args)
        {
            //输出结果原样
            String greeting = "     hello  world!      ";
            Console.WriteLine($"[{greeting}]");
            //输出结果，前面的空白被去掉了
            String trimmedGreeting = greeting.TrimStart();
            Console.WriteLine($"[{trimmedGreeting}]");
            //输出结果，后面的空白被去掉了，前面仍然保留
            trimmedGreeting = greeting.TrimEnd();
            Console.WriteLine($"[{trimmedGreeting}]");
            //前后空白都去掉了
            trimmedGreeting = greeting.Trim();
            Console.WriteLine($"[{trimmedGreeting}]");
        }
    }
}

还有一种是搜索字符串，就是在一堆字符串里搜自己想要的，然后返回布尔值

    string songLyrics = "You say goodbye, and I say hello";
    Console.WriteLine(songLyrics.Contains("goodbye"));
    Console.WriteLine(songLyrics.Contains("greetings"));
    //呃，以后为了省点页面长度，所以前面的都去掉了

另外字符串搜索还有个方法，叫StartsWith和EndWith，这两个方法会从开头或者结尾中分别去找字符串

    string songLyrics = "You say goodbye, and I say hello";
    Console.WriteLine(songLyrics.StartsWith("You")); 
    Console.WriteLine(songLyrics.StartsWith("goodbye"));

    Console.WriteLine(songLyrics.EndsWith("hello"));
    Console.WriteLine(songLyrics.EndsWith("goodbye"));
